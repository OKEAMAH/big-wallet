Big Wallet ist eine Open Source Krypto-Wallet mit einer Safari-Erweiterung

Verbinde eine Wallet auf jeder Website in Safari, indem du auf einen Metamask-Knopf klickst

Unterstützt Ethereum, Zora, Base, Polygon, Binance Smart Chain, Optimism, Arbitrum, Avalanche und mehr!

Wallets werden sicher in einem Schlüsselbund aufbewahrt
